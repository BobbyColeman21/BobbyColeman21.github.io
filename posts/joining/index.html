In this assignment, we used Nebraska Department of Education data to practice joining. Like all projects, I started by loading the library and data.

library(tidyverse)
library(janitor)
scores <- read.csv("NSCAS_ELA_Proficient_20232024.csv") |> clean_names()

This dataset is pretty big and to join it, I need to filter it so I am only left with the columns that I wanted. After doing this, I created the new dataset readingdistricts.

readingdistricts <- scores |> 
  filter(subgroup_type == "ALL STUDENTS") |> 
  filter(level == "DI") |> 
  filter(grade == "8") |> 
  filter(average_scale_score != "*") |> 
  select(level, school_year, county, district, name, subject, grade, average_scale_score)

I then loaded data about student discipline and used filter to get the same columns as I have in the above dataset.

discipline <- read.csv("StudentDiscipline_20232024.csv") |> clean_names()
districtdiscipline <- discipline |> 
  filter(subgroup_description == "All Students") |> 
  filter(level == "DI") |> 
  filter(students_disciplined_count != "*") |>
   select(level, school_year, county, district, name, total_student_population_count, students_disciplined_count)
